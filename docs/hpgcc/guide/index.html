Programming with

 

A Quick Start Guide.

 

This manual aims to give the user a brief introduction to programming the HP49G+ with HP-GCC. It is not intended to teach the C programming language. If you are new to C, there are many on-line C tutorials you can look at first.

 

This guide refers to features present in HPGCC Version 1.1.
